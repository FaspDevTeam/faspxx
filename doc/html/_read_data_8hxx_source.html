18 /*
19  * This class defines the basic MAT data structure and its basic operations. The
20  * CSRx data structure is an extension of the wellknown CSR sparse matrix format.
21  * The differences lie in the following two aspects:
22  * (1) Unlike the classical CSR format, the CSRx format requires the column
23  * indices in each row are in ascending order;
24  * (2) The CSRx format has a diagPtr array which stores the locations of the
25  * diagonal entries in each row.
26  *
27  * We give two simple examples here:
28  *
29  * Example 1.
30  * \begin{pmatrix}
31  * 1 & 0 & 2 \\
32  * 0 & 1 & 0 \\
33  * 3 & 0 & 0 \\
34  * \end{pmatrix}
35  * and
36  * nnz = 5, row = 3, col = 3,
37  * values = { 1, 2, 1, 3, 0 },
38  * colInd = { 0, 2, 1, 0, 2 },
39  * rowPtr = { 0, 2, 3, 5 },
40  * diagPtr = { 0, 2, 4 }.
41  *
42  * Example 2.
43  * \begin{pmatrix}
44  * 1& 7& 2 \\
45  * 0& 0& 0 \\
46  * 3& 0& 4 \\
47  * \end{pmatrix}
48  * and
49  * nnz = 6, row = 3, col = 3,
50  * values = { 1, 7, 2, 0, 3, 4 },
51  * colInd = { 0, 1, 2, 1, 0, 2 },
52  * rowPtr = { 0, 3, 4, 6 },
53  * diagPtr = { 0, 3, 5 }.
54  *
55  * Note that the CSRx format stores the diagonal entries even if they are zero.
56  * Furthermore, it is compatible with all CSR subroutines!
57  */
