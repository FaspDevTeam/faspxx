12/*
13 * This class defines the basic MAT data structure and its basic operations. The
14 * CSRx data structure is an extension of the wellknown CSR sparse matrix format.
15 * The differences lie in the following two aspects:
16 * (1) Unlike the classical CSR format, the CSRx format requires the column
17 * indices in each row are in a rough ascending order (namely, the nonzero
18 * entries should be ordered as lower trig, diag, uppper trig);
19 * (2) The CSRx format has a diagPtr array which stores the locations of the
20 * diagonal entries in each row.
21 * Note that the CSRx format stores the diagonal entries even if they are zero.
22 * Furthermore, it is compatible with all CSR subroutines!
23 */
