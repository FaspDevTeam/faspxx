19/*
20 * This class defines the basic MAT data structure and its basic operations. The
21 * CSRx data structure is an extension of the wellknown CSR sparse matrix format.
22 * The differences lie in the following two aspects:
23 * (1) Unlike the classical CSR format, the CSRx format requires the column
24 * indices in each row are in ascending order;
25 * (2) The CSRx format has a diagPtr array which stores the locations of the
26 * diagonal entries in each row.
27 *
28 * We give two simple examples here:
29 *
30 * Example 1.
31 * \begin{pmatrix}
32 * 1 & 0 & 2 \\
33 * 0 & 1 & 0 \\
34 * 3 & 0 & 0 \\
35 * \end{pmatrix}
36 * and
37 * nnz = 5, row = 3, col = 3,
38 * values = { 1, 2, 1, 3, 0 },
39 * colInd = { 0, 2, 1, 0, 2 },
40 * rowPtr = { 0, 2, 3, 5 },
41 * diagPtr = { 0, 2, 4 }.
42 *
43 * Example 2.
44 * \begin{pmatrix}
45 * 1& 7& 2 \\
46 * 0& 0& 0 \\
47 * 3& 0& 4 \\
48 * \end{pmatrix}
49 * and
50 * nnz = 6, row = 3, col = 3,
51 * values = { 1, 7, 2, 0, 3, 4 },
52 * colInd = { 0, 1, 2, 1, 0, 2 },
53 * rowPtr = { 0, 3, 4, 6 },
54 * diagPtr = { 0, 3, 5 }.
55 *
56 * Note that the CSRx format stores the diagonal entries even if they are zero.
57 * Furthermore, it is compatible with all CSR subroutines!
58 */
